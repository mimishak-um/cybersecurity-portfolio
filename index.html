Project Experience


Project 1: Implementing Automation to Ease SOC Workload

Objective: To reduce SOC analyst fatigue and streamline incident management processes by automating repetitive tasks and ticketing workflows.

Tools Used: Stellar Cyber SIEM, Jira Ticketing Systems.

Methodology:

  • Configured Stellar Cyber SIEM to generate enriched alerts by correlating logs from multiple sources.
  • Automated the creation of incident tickets by integrating SIEM-generated alerts with Jira ticketing system.
  • Created a rule-based system within the SIEM to filter out false positives, ensuring analysts only received actionable alerts.
  • Established automated reporting templates to provide daily and monthly security event summaries.

Outcome:

  • Reduced analyst workload by eliminating manual triaging and repetitive tasks.
  • Improved SLA adherence for faster resolution times for critical incidents.
  • Enhanced accuracy of escalated incidents, significantly reducing false positives and improving analyst satisfaction.

Project 2: Enhancing SOC Efficiency through Vulnerability Management

Objective: To improve an organization's security posture by identifying and remediating vulnerabilities across its network.

Tools Used: Rapid7 Nexpose

Methodology:

  • Conducted comprehensive vulnerability scans using Rapid7 Nexpose, identifying critical exposures.
  • Worked closely with the IT team to implement patch management and secure misconfigured assets.
  • Correlated vulnerability scan results with SIEM logs to prioritize remediation based on real-time threat activity.
  • Performed regular audits to ensure mitigated vulnerabilities remained resolved and introduced no new risks.

Outcome:

  • Successfully mitigated high-priority vulnerabilities in a short timeframe, significantly enhancing the organization's security posture.
  • Strengthened the organization's defense against external threats and improved compliance with security standards.
